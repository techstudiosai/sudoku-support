Politica de Privacidade

Ultima atualizacao: 17 de fevereiro de 2026

A Tech Studios AI ("nos") opera o aplicativo Sudoku Pro Genius ("App"). Esta pagina informa sobre nossas politicas relativas a coleta, uso e divulgacao de dados pessoais quando voce utiliza nosso App.

Dados que coletamos

Nos nao coletamos dados pessoais diretamente. O App nao exige cadastro, login ou qualquer informacao pessoal para funcionar. Todos os dados do jogo (progresso, estatisticas, configuracoes) sao armazenados localmente no seu dispositivo.

Servicos de terceiros

O App utiliza os seguintes servicos de terceiros que podem coletar dados:

  • Google AdMob - Utilizado para exibir anuncios. O Google pode coletar identificadores de dispositivo e dados de uso para personalizar anuncios. Consulte a Politica de Privacidade do Google.
  • Apple Game Center - Utilizado para leaderboards e rankings. Os dados sao gerenciados pela Apple conforme sua Politica de Privacidade.

Dados de publicidade

O Google AdMob pode coletar:

  • Identificadores de publicidade do dispositivo (IDFA)
  • Dados de interacao com anuncios
  • Informacoes gerais do dispositivo (modelo, sistema operacional)

Esses dados sao usados para exibir anuncios relevantes e nao sao vinculados a sua identidade pessoal por nos.

Dados de criancas

O App nao e direcionado a criancas menores de 13 anos. Nao coletamos intencionalmente dados de criancas.

Armazenamento local

Os seguintes dados sao armazenados apenas no seu dispositivo:

  • Progresso do jogo e jogos salvos
  • Estatisticas (jogos iniciados, vitorias, sequencias)
  • Configuracoes (tema, idioma, preferencias)
  • Apelido escolhido pelo usuario

Esses dados nao sao transmitidos para nossos servidores.

Alteracoes nesta politica

Podemos atualizar esta Politica de Privacidade periodicamente. Recomendamos revisar esta pagina regularmente.
